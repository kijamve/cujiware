Banco del Tesoro para WooCommerce - Pago Móvil y BT Pagos (C2P)

Integra los métodos de pago del Banco del Tesoro en tu tienda WooCommerce. Este plugin te permite aceptar pagos móviles y BT Pagos (C2P), ofreciendo una solución de pago rápida y confiable para tus clientes en Venezuela.

ADVERTENCIA CRÍTICA: No adquiera nuestra membresía sin haber iniciado los trámites administrativos con el banco. Los accesos al API son limitados a empresas jurídicas y están sujetos a condiciones confidenciales del banco. No ofrecemos devolución si el banco no acepta tu modelo de negocio o revoca los accesos al API.

Características Principales

Pago Móvil

Acepta pagos móviles de manera segura y eficiente, permitiendo a tus clientes realizar transacciones desde cualquier banco en Venezuela.

BT Pagos (C2P)

Integración con BT Pagos (C2P) para una experiencia de pago moderna y rápida, ideal para comercios que buscan expandir sus opciones de pago.

Validación Automática

Sistema de validación automática de pagos que confirma las transacciones de manera instantánea, mejorando la experiencia del usuario.

Panel de Control

Interfaz intuitiva para gestionar transacciones, configurar credenciales y monitorear el estado de los pagos en tiempo real.

Requisitos del Sistema

  • WooCommerce 4.0 o superior
  • PHP 7.0 o superior
  • Librerías Curl para PHP
  • Cuenta en el Banco del Tesoro
  • Credenciales de acceso al sistema de pagos

Beneficios para tu Negocio

Pagos Instantáneos

Recibe confirmación inmediata de los pagos, mejorando la eficiencia de tu operación.

Fácil Integración

Implementación sencilla y rápida en tu tienda WooCommerce existente.

Soporte Técnico

Asistencia técnica especializada para resolver cualquier duda o problema durante la implementación.
