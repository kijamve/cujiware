Banco BNC para WooCommerce - Solución de Pagos Oficial

Integra la pasarela de pago oficial del Banco Nacional de Crédito en tu tienda WooCommerce. Acepta tarjetas de crédito/débito Visa/Mastercard nacionales e internacionales, débito del BNC y Pago móvil (Clásico y C2P).

IMPORTANTE: La cancelación de tu membresía resultará en la pérdida de acceso a las funcionalidades del plugin.

ADVERTENCIA CRÍTICA: No compres este plugin sin haber iniciado los trámites administrativos con el banco. Los accesos al API son limitados a empresas jurídicas y están sujetos a condiciones confidenciales del banco. No ofrecemos devolución si el banco no acepta tu modelo de negocio o revoca los accesos al API.

Características Principales

Múltiples Métodos de Pago

Acepta tarjetas de crédito/débito Visa/Mastercard (nacionales e internacionales), débito del BNC y Pago móvil (Clásico y C2P).

Integración Oficial

Conexión directa con la API oficial del Banco Nacional de Crédito para transacciones seguras y confiables.

Experiencia de Usuario

Proceso de pago optimizado a través de aplicaciones móviles o portal web del banco.

Requisitos del Sistema

  • Librerías Curl y OpenSSL para PHP
  • PHP 7 o superior
  • WooCommerce instalado y configurado
  • Credenciales Sandbox o de Producción del API BNC
  • Empresa jurídica con trámites administrativos aprobados por el banco

Proceso de Implementación

Trámites Administrativos

Contacta con tu agencia comercial del BNC más cercana para iniciar el proceso de aprobación y obtener las credenciales necesarias.

Configuración Técnica

Una vez aprobado, configura el plugin con las credenciales proporcionadas por el banco.

Beneficios para tu Negocio

Soluciones de Pago Completas

Ofrece múltiples opciones de pago a tus clientes, adaptándose a sus preferencias.

Seguridad Garantizada

Procesamiento de pagos a través de la plataforma oficial y segura del banco.

Integración Perfecta

Conexión fluida entre tu tienda WooCommerce y la pasarela de pago del BNC.
