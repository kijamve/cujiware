Pasarela de Pago Binance Pay para PrestaShop: Cobros por QR con Validación Automática

Integra Binance Pay en tu tienda PrestaShop y ofrece a tus clientes una forma moderna y segura de realizar pagos con criptomonedas. Nuestro plugin automatiza todo el proceso de validación de pagos, eliminando la necesidad de verificación manual y agilizando tus transacciones.

Funcionalidades Clave

Pagos por QR Automatizados

Genera códigos QR únicos para cada transacción y recibe confirmaciones automáticas de pago. El sistema valida las transacciones sin intervención manual.

Soporte para Stablecoins

Acepta pagos en las principales stablecoins: USDT, BUSD, TUSD, USDC y DAI. Ofrece a tus clientes flexibilidad en sus opciones de pago.

Gestión de Reembolsos

Procesa reembolsos totales o parciales de manera sencilla y rápida directamente desde tu panel de administración.

Integración con Cuenta Merchant

Conecta fácilmente tu cuenta Merchant de Binance Pay para gestionar todos tus cobros de forma centralizada y segura.

Sistema de Proxy Integrado

Incluye un sistema de proxy para servidores ubicados en países restringidos, asegurando la compatibilidad y funcionamiento óptimo.

Validación Automática

Elimina la necesidad de verificación manual de pagos. El sistema confirma automáticamente las transacciones completadas.

Requisitos del Sistema

Requisitos Técnicos

  • PrestaShop 1.6.x / 1.7.x / 8.x
  • PHP 5.6 o superior (64 bits)
  • Servidor con hora sincronizada con la hora mundial
  • IP del hosting fuera de USA y países restringidos

Requisitos de Cuenta

  • Cuenta Merchant de Binance Pay (no se aceptan verificaciones P2P o trading)
  • Registro en merchant.binance.com

Beneficios para tu Negocio

Eficiencia Operativa

Reduce el tiempo de procesamiento de pagos y elimina la necesidad de verificación manual, optimizando tus operaciones diarias.

Seguridad Avanzada

Aprovecha la infraestructura de seguridad de Binance y el sistema de verificación independiente de Binance Pay.
