Banco Mercantil para Prestashop - Pasarela de Pago Completa

Integra la pasarela de pago del Banco Mercantil en tu tienda Prestashop. Este plugin te permite aceptar tarjetas de crédito, tarjetas de débito del Mercantil y pagos móviles C2P, ofreciendo una solución de pago completa y moderna para tus clientes.

IMPORTANTE: Este plugin requiere aprobación previa del Banco Mercantil. No compres si no has iniciado los trámites administrativos con el banco. Los accesos al API son limitados a empresas jurídicas y están sujetos a condiciones confidenciales del banco.

Características Principales

Múltiples Métodos de Pago

Acepta tarjetas de crédito de cualquier banco nacional e internacional, tarjetas de débito del Mercantil y pagos móviles C2P.

Botón de Pagos Mercantil

Implementación rápida del punto de venta virtual, facilitando la gestión de cobros en cualquier momento.

Integración API Oficial

Utiliza la API oficial del Banco Mercantil, asegurando compatibilidad y soporte continuo.

Panel de Administración

Interfaz intuitiva para gestionar transacciones, configurar credenciales y monitorear el estado de los pagos.

Requisitos del Sistema

  • Prestashop 1.6.x / 1.7.x
  • PHP 7.0 o superior
  • Librerías Curl para PHP
  • Cuenta Jurídica en el Banco Mercantil
  • Afiliación a Mercantil en Línea Empresas
  • Credenciales Sandbox o de Producción del Portal API
  • Aprobación previa del banco

Consideraciones Importantes

Proceso de Aprobación

Cada comercio debe gestionar su propio proceso de aprobación con el Banco Mercantil. Contacta a tu agencia comercial más cercana para iniciar los trámites necesarios.

Requisitos del Banco

Debes ser cliente jurídico del Mercantil, estar afiliado a Mercantil en Línea Empresas y disponer de una página web para la integración del Botón de Pagos.

Política de Reembolso

No se ofrecen reembolsos si el banco no acepta tu modelo de negocio o si los accesos al API son revocados posteriormente.

Beneficios para tu Negocio

Ampliación de Opciones de Pago

Ofrece a tus clientes múltiples opciones de pago, incluyendo tarjetas de crédito, débito y pago móvil C2P.

Implementación Rápida

Integración sencilla y rápida del Botón de Pagos Mercantil, facilitando la gestión de cobros en tu tienda online.

Solución Digital Avanzada

Forma parte del ecosistema de Open Banking del Banco Mercantil, adaptando tu negocio a la era digital.

Resumen

Optimiza tu tienda Prestashop con la pasarela de pago oficial del Banco Mercantil. Este plugin te ofrece una solución completa para la gestión de pagos, aceptando tarjetas de crédito nacionales e internacionales, débito del Mercantil y pagos móviles C2P. Con una implementación rápida y sencilla del Botón de Pagos Mercantil, mejora la experiencia de tus clientes y expande tus opciones de pago con la integración oficial del banco. ¡Potencia tu negocio en la era digital con una solución de pago moderna y confiable!
